About

Open Web Application Security Project(OWASP) is an international not-for-profit organization focused on improving the security of application software. Its mission is to make security aspects of software more visible in various ways including the production of extensive documentation of vulnerabilities, tools and technologies. In doing so, it aims to educate people about risks associated with commonly used web applications. Our Local Chapter Meetings are free and open to anyone to attend so both members and non-members are always welcomed.
